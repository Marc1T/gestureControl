Tester les nouveaux gestes :
Préparer :
Pièce bien éclairée, webcam à 50-100 cm.
Ouvre un navigateur ou une application pour tester (ex. page web pour défilement, dossier pour clics).
Mode main :
Déplacement : Main ouverte, déplace-la. Vérifie si le curseur suit.
Clic gauche : Pince pouce et index. Teste sur un bouton.
Clic droit : Pince pouce et majeur. Vérifie le menu contextuel.
Double-clic : Pince pouce-index deux fois. Teste sur une icône.
Glisser-déposer : Ferme le poing, déplace, ouvre la main. Teste avec un fichier.
Défilement : Lève index et majeur, déplace verticalement. Teste dans un navigateur.
Zoom : Écarte/rapproche les index des deux mains. Teste dans une visionneuse d’images.
Mode nez :
Clique sur "Passer au mode Nez".
Clic gauche : Cligne de l’œil. Teste sur un bouton.
Défilement : Hoche la tête. Teste dans un navigateur.
Mouvement : Déplace la tête. Vérifie si le curseur suit.
Chatbot :
Pose des questions comme "Comment défiler ?" ou "Quels sont les gestes ?".
